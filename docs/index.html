📝 Project Overview

This project implements a comprehensive Reinforcement Learning environment for training AI agents on classic Atari 2600 games. Built with modern RL libraries and best practices, it provides a complete setup for research and experimentation.

Key Features:

  • ✅ Full Atari 2600 game library (110+ games)
  • ✅ Pre-configured environment with standard preprocessing
  • ✅ Random and DQN agent implementations
  • ✅ Training and evaluation scripts with video recording
  • ✅ Comprehensive logging and visualization
  • ✅ Modular architecture for easy extension

Quick Start:

# Clone and setup
git clone https://github.com/yourusername/atari.git
cd atari
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# Test the environment
KMP_DUPLICATE_LIB_OK=TRUE python scripts/test_setup.py

# Train an agent
KMP_DUPLICATE_LIB_OK=TRUE python scripts/train.py --game Breakout --agent random --episodes 100
